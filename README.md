# Ajrly OS — منظومة أجرلي المتكاملة

نظام تشغيل موحّد لإدارة عمليات منصة **أجرلي** (Ajrly) لتأجير العقارات، يُحوّل ملف Google Sheets المتناثر إلى تطبيق ويب واحد متناسق، عالي الأداء، ثنائي اللغة (عربي/إنجليزي)، يعمل بانسجام بين فريق العمل والعملاء (ملاك العقارات).

> A unified operating system for the Ajrly property-rental platform. It consolidates a scattered, multi-tab Google Sheet into one coherent, high-performance, bilingual (AR/EN) web app.

---

## لماذا؟ — من الشيت إلى المنظومة

كان العمل موزّعاً على عدة تبويبات منفصلة داخل ملف واحد:

| تبويب الشيت | أين أصبح في المنظومة |
|---|---|
| `Asel Task Sheet` / `Kenda Task Sheet` (مهام لكل شخص) | **المهام** — لوحة موحّدة مع فلتر لكل عضو فريق |
| `Content Pillars` + جدول المحتوى | **استوديو المحتوى** + **الاستراتيجية** |
| `Property Owner Recruitment / Communications / Content` | **ملاك العقارات** — مسار (Pipeline) واحد بأربع مراحل |
| القيم، الأهداف، الركائز | **الاستراتيجية** |
| ملخصات الحالة والعدّادات اليدوية | **لوحة التحكم** — تُحسب آلياً |
| قالب Eisenhower الشخصي (بيانات وهمية) | حُذف — ضوضاء |

---

## الوحدات

1. **لوحة التحكم (Dashboard)** — مؤشرات حية: إجمالي المهام، الإنجاز، التوزيع حسب الحالة والفريق، المهام القادمة، توزيع المحتوى حسب الركيزة.
2. **المهام (Tasks)** — عرض لوحة (Kanban) بالسحب والإفلات + جدول، فلترة حسب عضو الفريق، أولويات وحالات موحّدة.
3. **استوديو المحتوى (Content Studio)** — تقويم أسبوعي + جدول، مربوط بالركائز والأهداف والمنصات.
4. **ملاك العقارات (CRM)** — مسار من 4 مراحل: استقطاب ← تواصل ← إنتاج محتوى ← نشط.
5. **الاستراتيجية (Strategy)** — روابط منصات أجرلي، القيم الأساسية، الأهداف، وركائز المحتوى العشر.

## المزايا التقنية

- **أداء عالٍ:** صفحة واحدة (SPA) بدون أي تبعيات بناء — تحميل فوري.
- **ثنائي اللغة + RTL/LTR** كامل، تبديل لحظي.
- **وضع داكن/فاتح**، تصميم متجاوب للجوال.
- **حفظ محلي** (localStorage) — بيانات مبدئية مستخرجة من الشيت الأصلي.

## التشغيل

لا يحتاج بناء. افتح `index.html` مباشرة، أو شغّل خادماً محلياً:

```bash
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```

النشر على Netlify يعمل تلقائياً عبر `netlify.toml` (موقع ثابت، بدون أمر بناء).

## خارطة الطريق

- **المرحلة 1 (الحالية):** الواجهة الكاملة + الوحدات الأربع + حفظ محلي. ✅
- **المرحلة 2:** قاعدة بيانات سحابية (Supabase/Postgres) + مصادقة وصلاحيات حسب الدور.
- **المرحلة 3:** استيراد بيانات الملاك تلقائياً من موقع أجرلي + تنبيهات التواصل.
- **المرحلة 4:** تحليلات وتقارير متقدمة + تكامل واتساب/البريد.
